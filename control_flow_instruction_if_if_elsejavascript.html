Control=Seguence of execution of instruction in a code is called control flow instruction.Basically by default it is from top-bottom but we can control it by using control flow instruction

Selection based control flow instruction

1. if

It should be associated with given condition.if condition is true then it should be executed,if it false then it should not be executed

Find a number it is prime or not

2. if---else

It should be associated with given condition if the condition in if scope is true then it should be executed,if the condition in if scope is false then it should not be executed

Program to find a number is prime or not
